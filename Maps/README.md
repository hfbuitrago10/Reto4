# Mapas :earth_americas:

Para la creación de mapas interactivos de conectividad se utilizó la librería ```folium``` junto con ```OpenStreetMap```. Cada mapa representa una versión aproximada de las conexiones de cables entre distintos puntos de conexión. A continuación, se describen los mapas del repositorio:

* ```mostconnectedmap:``` este mapa presenta las conexiones del punto de conexión con mayor número de cables conectados.
* ```mincostpathmap:``` este mapa presenta la ruta de distancia mínima desde Bogotá, Colombia hacia Jakarta, Indonesia.
* ```mstshortlongestmap:``` este mapa presenta las conexiones de mayor y menor distancia del árbol de expansión minima de la red.
* ```affectedcountriesmap:``` este mapa presenta las conexiones del punto de conexión Fortaleza.

Los mapas pueden ser visualizados [aquí.](https://jsfiddle.net)
